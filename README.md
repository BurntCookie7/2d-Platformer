How to play:
  - Have at least python 3.13.14 installed (may work on other versions but that is what I have installed and am using)
  - Have the latest version of the pygame module for python installed (may work on other versions but that is what I have installed and am using)
  - download the latest release with all the textures and map files
  - just run the .py file in the same folder that it was downloaded in and enjoy!
